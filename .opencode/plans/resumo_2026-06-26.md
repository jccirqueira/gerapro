## Notas Técnicas

### Fluxo de renderização
1. Usuário clica botão → `_showLayoutConfigPanel(cabId)`
2. Lê `eq` do store (`data?.equipments?.[this.activeEquipmentIndex]`)
3. `const dflt = this._getDefaultLayoutConfig()`
4. `const lc = face ? (...) : (cab.layoutConfig || dflt)`
5. Renderiza HTML com `can.id` nos onclick dos botões
6. Quando clica Edit → `_editarCanaleta(canId, cabId)`
7. Chama `_getCanaletaTarget(cabId)` que lê o mesmo `eq` e retorna `cab.layoutConfig` (ou cria um novo se null)

### Função `_getCanaletaTarget(cabId)`
```js
_getCanaletaTarget(cabId) {
    const data = store.getState();
    const eq = data?.equipments?.[this.activeEquipmentIndex];
    const actualCabId = cabId.replace(/_(face[12])$/, '');
    const cab = eq.layoutConfig?.cabinetAssignments?.[actualCabId];
    if (!cab) return this._getDefaultLayoutConfig();
    if (!cab.layoutConfig) cab.layoutConfig = this._getDefaultLayoutConfig();
    return cab.layoutConfig;
}
```
